Core Features

Built as a complete student community product

FriendZone combines social networking, real-time communication, recommendation logic, and AI assistance in one modern platform.

🧠

Personality-Based Matching

A 24-question MBTI-style test helps the system understand each student’s social profile.

🎯

Smart Community Assignment

The matching engine combines hobbies, tags, categories, and personality affinity.

💬

Real-Time Chat

Community rooms support instant messaging, typing indicators, and emoji reactions.

🤖

GPT / Fallback Assistant

The AI assistant generates topics, event ideas, icebreakers, and group activities.

🏫

Inter-University Vision

Designed to expand beyond one campus into city-wide and country-wide student networks.

📊

Admin Dashboard

Admins can track users, communities, messages, personality stats, and popular hobbies.

How It Works

From signup to meaningful community interaction

FriendZone turns onboarding into a social discovery flow. Each step improves the quality of community recommendations.

01

Register with a university email

Only `.edu.tr` addresses are accepted to keep the platform student-focused.

02

Complete the personality test

The system calculates an MBTI-like personality type for matching.

03

Select hobbies and interests

Hobbies are mapped to categories and compared against community tags.

04

Join matched communities

Students can chat, react, discover people, and receive AI-powered group suggestions.

FriendZone 2.0 Vision

From campus matching to a nationwide student community network

The next version of FriendZone is designed to support profile settings, persistent profile photos, university and city-based discovery preferences, community events, posters, participation, reviews, and inter-university collaboration.

Profile & Settings Photo upload, discovery preferences, bio, city, and visibility scope.
Events & Posters Communities can create events with posters, capacity, location, and date.
Reviews & Feedback Students can rate and review events after participating.
Cross-University Discovery Students can discover communities from their university, city, or country.

Tech Stack

Designed for full-stack, AI, and cloud interview readiness

Frontend

HTML5, CSS3, Vanilla JavaScript, Socket.IO Client, Chart.js, responsive dark UI

AI / Matching

Weighted recommendation engine, MBTI affinity, hobby-category mapping, OpenAI fallback architecture

Deployment

Docker, Docker Compose, PostgreSQL container, Nginx frontend container, environment configuration
